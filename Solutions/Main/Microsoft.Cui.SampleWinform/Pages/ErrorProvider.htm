Description

This demonstration displays how an ErrorProvider can be added to any of the input controls listed below:

  • DateInputBox
  • TimeInputBox
  • TimeSpanInputBox

In addition to these three basic examples, an example of how a developer may choose to add a customised ErrorProvider to the controls has been given. This example displays how a developer can add contextual logic to the controls or even simply ensure that the ErrorProviders are aligned in look-n-feel to the rest of their application.
