Description

The TimeLabel control lets you display a time in a clear and unambiguous way. The TimeLabel control may include:

  • An exact time
  • A time marked as approximate
  • A time marked as ongoing
  • A null index time where the exact text displayed can be defined by the Independent Software Vendor. For example, a time marked as "Ongoing".

Additional information relating to this control can be found on the corresponding MSCUI control page.
